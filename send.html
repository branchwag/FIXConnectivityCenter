Single message
FIX fields are tag=value separated by |, ^, or SOH. JSON keys are FIX tags (or MsgType). Header fields (8/9/10/34/49/52/56) and Sender/Target are filled from the session.
CSV batch
One FIX message is sent per data row, in order.
    Accepted format

    The first row is a header of column names; each following row is one message.

    • MsgType — the FIX message type (tag 35), e.g. D for New Order – Single.
    • Numeric columns are FIX tags carried in the message body, e.g. 55 Symbol, 54 Side, 38 OrderQty, 44 Price.
    • SenderCompID / TargetCompID columns are optional and ignored — they're filled from the selected session.
